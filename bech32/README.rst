Cardano bech32
==============================================================================

Haskell implementation of the Bech32 address format (BIP 0173). 

A small executable is available for rapid conversion in the console. 
You can obtain a pre-compiled bech32 binary for Linux, macOS, and 
Windows as part of the cardano-wallet
